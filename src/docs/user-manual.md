# User Manual for Doctor E-Medicine Platform

## Overview
This platform allows doctors and patients to manage appointments, view availability, and handle medical records.

## Registration
1. Navigate to the login page.
2. Provide your email and password to sign up.
3. Choose your role (Doctor or Patient).

## Appointment Booking
1. Doctors can manage their availability slots.
2. Patients can browse available slots and book appointments.
3. Appointment status updates are shown on the dashboard.

## Admin Functions
1. Admin can add/remove users.
2. Admin can view user activity and appointments.
